| Orientada a Eventos | NotificationComponet.java | Son eventos que esperan la activacion mediante TransmisionListener que funciona como manejador dispare la accion | No se puede convertir a OOP puro ya que se deberia de eliminar transmisionListener e instanciar de manera directa las notificaciones |
| Funcional | StatsService.java | Es funcional ya que utiliza funciones de una libreria stream.Collectors que permite la obtencion de datos sin modificar datos | Para cambiarlo a poo puro consiteria en la eliminacion de la libreria stream.collectos y se tendria que implementar en el metodo una variable contador con estructuras de bucle y devuelva la cantidad |
| Estructurada | StatsService.java | Utiliza metodos static que funcionan como funciones libres, no tiene variables de instancia solo recibe datos y los procesa linea a linea| Quitar la palabra statics de los metodos, instanciar variables para que guarde los guarde como un atributo interno |
